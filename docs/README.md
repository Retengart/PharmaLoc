# Геомаркетинговый анализ для размещения аптек

> **Определение оптимальных локаций для размещения аптек с использованием методов машинного обучения и пространственных данных OpenStreetMap**

## 📋 Описание проекта

Проект реализует комплексный геомаркетинговый анализ для определения перспективных территорий для открытия аптек. Система использует данные OpenStreetMap, гексагональную сетку H3 и ансамблевые методы машинного обучения для оценки потенциала территорий.

### Ключевые возможности

- 🗺️ **Сбор геопространственных данных** из OpenStreetMap (аптеки, медицинские учреждения, транспорт, жилье, коммерция)
- 🔷 **Гексагональная сетка H3** для равномерного деления территории (разрешение 9)
- 📊 **Feature Engineering**: 50+ пространственных признаков (расстояния, плотности, покрытия)
- 🤖 **Машинное обучение**: Baseline (Logistic Regression), Random Forest, CatBoost с оптимизацией через Optuna
- 📦 **Кластерный анализ**: K-means с автоматическим подбором числа кластеров
- 📈 **Комплексная визуализация**: ROC/PR кривые, Confusion Matrix, t-SNE/PCA, профили кластеров
- 🗺️ **Интерактивные карты**: Folium с тепловыми картами и детальными popup

## 📊 Pipeline анализа

### Этап 1: Сбор данных
- Определение границ ROI (СВАО Москвы по умолчанию)
- Параллельная загрузка данных из OSM (ThreadPoolExecutor)
- Сохранение в GeoJSON формате

### Этап 2: Feature Engineering
- Создание гексагональной сетки H3 (разрешение 9)
- Расчет дистанционных признаков (SciPy cKDTree, O(log N))
- Расчет площадных признаков (STRtree + Pandarallel)
- Анализ конкурентной среды (сетевые vs одиночные аптеки)

### Этап 3: EDA
- Матрица корреляции признаков
- Распределение целевой переменной (анализ дисбаланса)
- Гистограммы признаков
- Box plots по классам

### Этап 4: Кластерный анализ
- Метод локтя + Silhouette Score для выбора K
- K-means кластеризация
- Визуализация: PCA, t-SNE
- Автоматическое описание профилей кластеров

### Этап 5: Машинное обучение
- **Baseline**: Logistic Regression
- **RandomForest**: Оптимизация через Optuna (50 trials)
- **CatBoost**: Оптимизация через Optuna (50 trials)
- SMOTE для балансировки классов
- 5-fold кросс-валидация

### Этап 6: Оценка качества
- ROC и PR кривые для каждой модели
- Confusion Matrix (абсолютная и нормализованная)
- Сравнительный график метрик
- Анализ важности признаков

### Этап 7: Рекомендации
- Расчет интегрального потенциала
- Формула: `0.6 × Взвешенная оценка + 0.4 × ML предсказание`
- Топ-10 локаций с минимальным расстоянием до конкурентов 300м
- Детальные отчеты в JSON и CSV

### Этап 8: Визуализация
- Интерактивная карта потенциала (Folium)
- Слои: тепловая карта, сетка H3, конкуренты, рекомендации
- Детальные popup с характеристиками локаций

## 📚 Методология

### Формула интегрального потенциала

```
Потенциал = 0.6 × (
    w_med × Синергия_медицина +
    w_subway × Доступность_метро +
    w_ground × Доступность_транспорт +
    w_res × Плотность_жилья +
    w_comm × Коммерч_активность +
    w_off × Плотность_офисов +
    w_park × Доступность_парковок +
    w_ped × Пешеход_доступность +
    w_comp × (-Конкуренция)
) + 0.4 × ML_предсказание
```

### Метрики качества

- **F1-Score** — основная метрика (баланс precision/recall)
- **ROC AUC** — качество ранжирования
- **Cross-validation** — устойчивость модели
