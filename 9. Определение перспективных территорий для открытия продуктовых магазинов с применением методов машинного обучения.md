ЗАДАНИЕ

НА ВЫПОЛНЕНИЕ ИТОГОВОЙ АТТЕСТАЦИОННОЙ РАБОТЫ

1. Тема итоговой аттестационной работы: Определение перспективных территорий для открытия продуктовых магазинов с применением методов машинного обучения

2. Исходные данные к итоговой аттестационной работе:

Пространственные данные из OpenStreetMap о продуктовых магазинах, супермаркетах и другой торговой инфраструктуре

Библиотеки для геопространственного анализа (osmnx, geopandas, h3, leafmap)

Библиотеки машинного обучения (scikit-learn) для построения прогнозных моделей

Документация по работе с гексагональной системой координат H3

Данные о демографических характеристиках территорий (при наличии)

3. Основные части итоговой аттестационной работы (перечень подлежащих разработке вопросов):

3.1. Сбор и подготовка геопространственных данных:

Определение границ исследуемой территории города/района (ROI)

Загрузка из OpenStreetMap данных о существующих продуктовых магазинах (теги: shop=supermarket, shop=convenience, shop=grocery и другие)

Извлечение данных о жилых зданиях, остановках транспорта, парковках, конкурентах

Проецирование всех данных в единую метрическую систему координат для корректного расчета расстояний

3.2. Создание территориальных единиц анализа и инженерия признаков:

Построение гексагональной сетки H3 с оптимальным разрешением (8-9 уровень)

Расчет плотности населения по количеству и характеристикам жилых зданий в радиусах 150м, 300м, 500м (или других, в зависимости от Ваших гипотез)

Подсчет транспортной доступности (остановки, станции метро) в зоне пешей доступности

Анализ конкурентной среды: количество существующих магазинов в различных радиусах

Расчет доступности парковочных мест и пешеходных зон

Нормализация и масштабирование всех признаков

3.3. Применение методов машинного обучения:

Формирование целевой переменной на основе существующих успешных локаций магазинов

Обучение ансамблевых моделей классификации: RandomForest, GradientBoosting

Настройка гиперпараметров через GridSearchCV

Кросс-валидация и оценка качества по метрикам ROC AUC, F1-score

Анализ важности признаков для выявления ключевых факторов успешности локации

3.4. Кластерный анализ территорий:

Определение оптимального числа кластеров методом локтя и SilhouetteScore

Применение K-means для выявления типов территорий по торговому потенциалу

Характеристика каждого кластера по доминирующим признакам

Интеграция кластерных меток как дополнительного признака в модель

3.5. Расчет интегрального показателя потенциала:

Оценка привлекательности каждой локации на основе вероятностных предсказаний модели

Учет фактора конкуренции через взвешенный показатель насыщенности территории

Формулировка формулы расчета (У нас на паре была сформирована такая Потенциал = 0.7 × Привлекательность + 0.3 × (1 - Конкуренция), у Вас будет своя, в зависимости от результатов Ваших экспериментов)

Категоризация территорий по уровню потенциала (высокий/средний/низкий)

3.6. Визуализация результатов и формирование рекомендаций:

Создание интерактивных карт с тепловым слоем потенциала территорий

Выделение топ-10 локаций без прямых конкурентов в радиусе 300м

Маркировка существующих магазинов и рекомендуемых точек размещения

Статистический анализ характеристик наиболее перспективных территорий

Экспорт координат рекомендуемых локаций и их характеристик в табличном формате

Подготовка аналитического отчета с обоснованием выбора локаций
